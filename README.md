# 🏆 Análise da Premier League com Power BI & PostgreSQL

## 📊 Visão Geral
Este projeto apresenta uma análise detalhada da Premier League, utilizando **Power BI** para visualização de dados e **PostgreSQL** para manipulação e consulta das informações. O dashboard inclui estatísticas essenciais sobre o desempenho dos times ao longo dos anos, fornecendo insights valiosos.

## 🔧 Tecnologias Utilizadas
- **Power BI**: Desenvolvimento de dashboards interativos `premier.pbix`
- **PostgreSQL**: Armazenamento e consulta dos dados `queries/consultas.sql`
- **DAX (Data Analysis Expressions)**: Cálculo de métricas e KPIs `dax/formulas_dax.txt`

## 📝 Recursos do Dashboard
- 🌟 **Total de partidas e gols** ao longo dos anos
- 📈 **Classificação dos times** com base em pontos, vitórias e empates
- 📊 **Distribuição de resultados** (vitórias, derrotas e empates)
- 🌟 **Comparativo entre vitórias em casa e fora**
- 🔍 **Filtro dinâmico de times**: É possível filtrar um ou mais times que estiveram na Premier League entre 1993 até 02/2025 a qualquer momento
